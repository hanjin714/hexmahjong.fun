# 海克斯麻将 H5

这是 `HexMahjong` 的 H5 原型版本，保留核心三层理念：

- 城邦：开局确定全局规则。
- 海克斯：自走棋式阶段强化。开局三选一，之后第 3、6 轮再次三选一，最多持有 3 个。
- 羁绊：根据手牌结构触发额外得分与胡牌反馈。
- 回合倒计时：玩家回合默认 18 秒，超时自动托管出牌。
- 动态操作：胡牌、活力、拆牌只在当前手牌满足判定时出现，减少误触。
- 操作提示：可胡、可拆、活力弃牌会在底部给出结果说明，并在对应手牌上高亮。
- 音效：使用 WebAudio 程序化生成点击、摸牌、出牌、海克斯、倒计时和胜负反馈音。
- 胜负结算：胡牌后弹出胜利/惜败结算层，展示牌型、分数、羁绊、最终手牌和粒子动效。
- 牌面体验：Canvas 程序化绘制筒/条/万牌面，混子显示具体底牌，并在移动端支持轻触感反馈。

## 本地运行

```bash
python3 -m http.server 5177
```

然后打开：

```text
http://localhost:5177/HexMahjongH5/
```

## 适配方向

- 普通网页：当前版本可直接运行。
- 微信小程序：可先用 WebView 承载，也可以把 `game.js` 的状态机迁移到小程序 Canvas。
- 微信小游戏：渲染层已经集中在 Canvas，后续替换 DOM 按钮和事件入口即可。
